# Freelance Gig Management System

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- HTML
- CSS
- Bootstrap 5
- JavaScript (ES6)

## Features
- User Registration
- User Login (JWT Authentication)
- Create Gig
- View All Gigs
- Edit Gig
- Delete Gig

## Installation

```bash
npm install
```

```bash
npm run dev
```

Open:

```
http://localhost:5000
```
