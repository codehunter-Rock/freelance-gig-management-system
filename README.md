# Freelance Gig Management System

A Full Stack Freelance Gig Management System developed using Node.js, Express.js, MongoDB, HTML, CSS, Bootstrap, and JavaScript. The application allows users to register, log in securely using JWT authentication, and manage freelance gigs through complete CRUD operations.

---

## 🚀 Features

- User Registration
- User Login with JWT Authentication
- Password Encryption using bcrypt
- Create New Gig
- View All Gigs
- Update Existing Gig
- Delete Gig
- RESTful APIs
- MVC Architecture
- Responsive Bootstrap UI
- MongoDB Atlas Database

## ⚙️ Installation

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
PORT=5000
MONGO_URI=YOUR_MONGODB_URI
JWT_SECRET=YOUR_SECRET_KEY
```

Run the project

```bash
npm run dev
```

Open your browser

```
http://localhost:5000
```
